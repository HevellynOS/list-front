codigos

Aqui temos os codigos importantes para implementar seu css

    Regra At

    Uma regra at (at-rule) é uma CSS statement iniciando com um caractere arroba, '@', seguido por um indetificador e terminando no primeiro ponto e vírgula';', fora de um bloco, ou no fim do primeiro Bloco CSS.

  1. @charset: definindo o conjunto de caracteres (codificação) usado pela folha de estilos.
  2. @import: informando o motor CSS para incluir uma folha de estilos externa.
  3. @media: um grupo de regras condicional que é aplicado ao conteúdo caso o dispositivo de exibição cumpra com a condição de mídia definida.
  4. @font-face: descrevendo o aspecto de uma fonte externa que deve ser baixada.
  5. @document: um grupo de regras condicional que é aplicado ao conteúdo caso o documento cumpra com os critério definidos.
  6. @supports: um grupo de regras condicional que é aplicado caso o navegador cumpra com os critérios definidos.
