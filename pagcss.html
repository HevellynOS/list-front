cascading style sheets :

CSS é um mecanismo para adicionar estilos, cores, fontes, espaçamento a uma página web ou aplicativos, aplicado diretamente nas tags HTML ou ficam contido dentro das tags style, um programador css tem como responsabilidade saber o minímo de design, como:

  1. Psicologia das Cores
  2. Hexadecimal, RGB, HSL
  3. Tipografia

listas de codigos

codigos de comando:

@charset "UTF-8"

formatação importante para não ter erros de acentuação

@font-face

importar font de dentro do seu computador

@import url

importar de algum site especifico para o seu projeto
